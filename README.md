# AI-Powered Product Assistant for Amazon Using Ollama

## Project Overview

This project is an AI-powered Product Assistant that combines the Amazon Products dataset with a locally hosted Large Language Model (LLM) using Ollama.

The assistant allows users to ask product-related questions in natural language. The application retrieves relevant information from the dataset using Pandas, then uses Ollama to generate a natural conversational response.

---

## Features

- Search products by maximum price
- Display top-rated products
- Display the most-reviewed products
- Display best-selling products
- Natural language understanding using Ollama
- Conversational responses generated by a local LLM

---

## Technologies Used

- Python
- Pandas
- Ollama
- Llama 3.2

## Installation

1. Clone the repository

```bash
git clone https://github.com/AbdulmlikY/AI-powered-Product-Assistant-For-Amazon-Using-Ollama_3.2.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Make sure Ollama is installed.

4. Download the model

```bash
ollama pull llama3.2
```

5. Run the application

```bash
python main.py
```

---

## How It Works

1. The user enters a question in natural language.
2. Ollama classifies the user's intent and returns a JSON object.
3. Python reads the JSON response.
4. Pandas searches the dataset using the appropriate search function.
5. The search results are sent back to Ollama.
6. Ollama generates a natural conversational response based only on the retrieved products.
